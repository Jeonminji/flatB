# 역할분담
- 팀장: 전민지<br>
- 팀원: 유인희, 이민주, 전현정

- 유인희
  - 주제 발표 PPT 제작, 중간 발표 보고서 작성
  - 회원가입(Back) 구현
  - 비교페이지(Back) 구현
- 이민주
  - 주제 발표, 중간 발표 발표자
  - 로그인(Front) 구현
  - 비교페이지(Front) 구현
- 전민지
  - 주제 발표 보고서 작성, 중간 발표 보고서 작성
  - 로그인(Back) 구현
  - 신고페이지(Back) 구현
- 전현정
  - 주제 발표 PPT 제작, 중간 발표 PPT 제작
  - 회원가입(Front) 구현
  - 신고페이지(Front) 구현
